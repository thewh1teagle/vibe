Vibe — design directions

Two groups, same component structure. The first group varies only the accent color. The second group varies the entire visual language.

GROUP A

Color variations

Same dark-premium language, different accent
01

Local · Cyan

Linear / Raycast reference. Clean and developer-focused.

02

Local · Indigo

Cooler indigo accent. Slightly more "premium SaaS" feel.

03

Local · Emerald

Emerald accent that matches the existing "Local" green badge.

04

Groq Cloud · Amber

The Groq / API path. Orange-amber accent, "API" indicator on the badge, Raw output toggle disabled with an "N/A · API" hint.

GROUP B

Unique visual languages

Same structure, completely different design vocabularies
05

Brutalist Mono

Sharp corners, monospace, B&W with one acid-green accent. Strict developer's tool.

06

Phosphor CRT

Green phosphor on black, scanlines, monospace, blinking-cursor terminal aesthetic.

07

Glassmorphism (dark)

Real frosted glass panels over a vibrant magenta/indigo/cyan aurora. Color shows through.

08

Blueprint

Technical drawing aesthetic — grid background, crosshair markers, dimension callouts, blueprint blue.
